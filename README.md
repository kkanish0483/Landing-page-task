# Mars Colony Recruitment

This project is a responsive landing page for a futuristic Mars settlement recruitment campaign.

## What was built

- A full HTML5 landing page with semantic sections: header, nav, hero, features, statistics, reasons, contact, and footer.
- A modern CSS3 design using Flexbox, CSS Grid, media queries, gradients, shadows, and glassmorphism.
- A responsive layout that adapts for desktop, tablet, and mobile screens.

## Files

- `index.html` - page structure and content
- `style.css` - all styling and responsive layout rules

## Notes

- No JavaScript was used.
- No external frameworks or libraries were used.
- The design follows a futuristic Mars / space-tech startup theme.
