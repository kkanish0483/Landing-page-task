# 🚀 Mars Colony Recruitment

A futuristic and fully responsive landing page designed to recruit the first generation of settlers for a human colony on Mars. Inspired by modern space-tech companies and interplanetary exploration concepts, this project demonstrates responsive web design using only HTML5 and CSS3.

## 🌌 Project Overview

Mars Colony Recruitment is a concept landing page that invites users to become one of the first 10,000 Martians. The website showcases the mission, colony lifestyle, eligibility requirements, and future vision of establishing a sustainable civilization on Mars.

The project was created as part of a web development task to demonstrate proficiency in HTML structure, CSS styling, Flexbox, Grid layouts, and responsive design principles.

## ✨ Features
